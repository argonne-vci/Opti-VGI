Installation
============

To install Opti-VGI, you can use pip:

.. code-block:: bash

   pip install optivgi

For development, clone the repository and install with development dependencies:

.. code-block:: bash

   git clone https://github.com/argonne-vci/Opti-VGI.git
   cd Opti-VGI
   pip install -e ".[dev]"
